Test Share and Rooms Language Switching

Test Pages

Share/Transfer Page Rooms Page Rooms Page (Standalone)

Instructions

  1. 点击上面的链接打开页面
  2. 尝试切换语言选择器
  3. 检查页面内容是否翻译
  4. 检查浏览器控制台的日志

Expected Console Logs

你应该在控制台看到类似这样的日志：

Language changed to: zh-CN
DROPSHARE_I18N.changeLanguage called with: zh-CN
translatePage called
Found X elements to translate
Translated nav_home to 首页
...

Debug Info

如果语言切换不工作，请检查：

  • 浏览器控制台是否有错误
  • 是否有 "Language changed to:" 日志
  • 是否有 "translatePage called" 日志
  • 语言选择器的值是否改变
